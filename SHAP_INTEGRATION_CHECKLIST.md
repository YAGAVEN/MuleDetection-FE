## API Endpoints ✓

### SHAP Endpoints
```
✓ GET /api/shap/model-reports?limit=50
  - Returns: List of suspicious accounts with SHAP explanations
  - Response: {status, total_suspicious, total_reports, showing, reports}

✓ GET /api/shap/model-reports/{account_id}
  - Returns: Detailed explanation for single account
  - Response: {status, report}

✓ POST /api/shap/generate-reports
  - Returns: Report generation status
  - Response: {status, reports_generated, total_suspicious}
```
